# tailwind-gen-config-app

用來設定 TailwindCss Config 的後台

## Features

目前能夠客製化設定的選項有:

- screens (break point)
- opacity
- colors
- spacing
- width
- font
  - font-size
  - font-weight
  - line-height
  - letter-spacing

**輸出檔案:** 為 __JSON__ 檔
